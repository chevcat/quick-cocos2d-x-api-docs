quick-cocos2d-x API 文档

API 文档是针对 quick-cocos2d-x 的 Lua framework 而言，底层的 C++ API 文档，请参考 cocos2d-x.org。

Lua framework 的基础模块由以下几个部分组成：

  • framework.functions 基础函数库
  • framework.debug 调试支持
  • framework.device 提供设备相关属性的查询，以及设备功能的访问
  • framework.display 与显示图像、场景有关的功能
  • framework.audio 播放音乐、音效
  • framework.ui 创建和管理用户界面
  • framework.transition 为图像创造效果
  • framework.network 网络服务
  • framework.crypto 加解密、数据编码
  • framework.json JSON 编码与解码
  • framework.luaoc Lua 与 Objective-C 的交互接口，仅用于 iOS
  • framework.luaj Lua 与 Java 的交互接口，仅用于 Android
  • framework.scheduler 全局计时器、计划任务

这些模块除了 framework.scheduler 之外，都会在游戏初始化时自动加载。
