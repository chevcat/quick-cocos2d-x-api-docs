framework.crypto

加解密、数据编码

~~

crypto.encryptAES256

使用 AES256 算法加密内容。

格式： 

ciphertext = crypto.encryptAES256(明文字符串, 密钥字符串)

提示： AES256 仅在 iOS 和 Mac 平台可用。加解密尽量选择跨平台的 XXTEA 接口。

~~

crypto.decryptAES256

使用 AES256 算法解密内容。

格式： 

plaintext = crypto.decryptAES256(加密后的字符串, 密钥字符串)

~~

crypto.encryptXXTEA

使用 XXTEA 算法加密内容。

格式： 

ciphertext = crypto.encryptXXTEA(明文字符串, 密钥字符串)

~~

crypto.decryptXXTEA

使用 XXTEA 算法解密内容。

格式： 

plaintext = crypto.decryptXXTEA(加密后的字符串, 密钥字符串)

~~

crypto.encodeBase64

使用 BASE64 算法编码内容。

格式： 

encoded = crypto.encodeBase64(原文字符串)

~~

crypto.decodeBase64

使用 BASE64 算法解码内容。

格式： 

decoded = crypto.decodeBase64(编码后的字符串)

~~

crypto.md5

计算内容的 MD5 码。

格式： 

md5code = crypto.md5(内容字符串, [是否返回二进制 MD5 码])

默认情况下，md5() 计算 MD5 码，并以十六进制字符串的形式返回。如果第二个参数为 true，则返回 MD5 码的二进制内容。
